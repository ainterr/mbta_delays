MBTA Delays

The MBTA wastes a lot of my time on my daily hour and a half commute due to it's aging, horribly maintained infrastructure. This is a way of quantifying all that lost time and comparing the MBTA to other methods of transportation that I could (and probably should) have taken.

I have spent 0 minutes commuting on the MBTA since starting this experiment.

Data Collection

This data was collected using a Raspberry Pi running a custom REST API written with Flask that fetches data from Google's distancematrix API. I'll write a blog post explaining the server (and how I make requests from my phone) some time soon.

I am not affiliated with the MBTA in any way. This data is a reflection of my personal experience - gathered using precise location data and Google Maps predictions for various transportation modes.
